## Instructions

Clone the repository
```bash
$ git clone https://github.com/rawfour/coffee-shop-spa.git
```

Move into the folder

```bash
$ cd .\weather-now\
```

Install Node.js and NPM.

```
https://nodejs.org/en/
```

Install dependencies
```bash
$ npm install
```

Run the application
```bash
$ npm run serve
```
